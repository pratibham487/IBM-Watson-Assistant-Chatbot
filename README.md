# IBM Watson Assistant Chatbot 🤖

## Overview

This project was developed as part of the IBM Generative AI Summer Training using IBM Watson Assistant and IBM Cloud.

The chatbot was designed to simulate a customer support assistant for a logistics and delivery platform. It guides users through delivery booking, order tracking, payment selection, and order confirmation using conversational AI.

---

## Features

- 📦 Delivery booking
- 🚚 Order tracking
- 💳 Payment method selection
- ✅ Order confirmation
- 💬 Conversational AI using IBM Watson Assistant

---

## Technologies Used

- IBM Watson Assistant
- IBM Cloud
- Natural Language Processing (NLP)
